# Skills

Skills for this project are deployed directly to `.claude/skills/` for Claude Code to discover on demand.

See `.claude/skills/` for all available skills:

- **git-flow** — Git branching, commit conventions, and merge strategy
- **code-quality** — Coding standards, file structure, and implementation conventions
- **research-document** — Format for research phase output
- **spec-document** — Format for specification phase output
- **design-document** — Format for design phase output
- **validation-report** — Format for validation phase output, validation scripts, and demo scripts
